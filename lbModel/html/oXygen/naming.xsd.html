Super-types: lgBuiltin:tsCaseIgnoreIA5String < URNMap (by extension) < supportedSource (by extension)
Sub-types: None
Name supportedSource
Abstract no
Documentation The URI and local name of an external source reference. Source references can also carry an additional compositional rule section that describes how to combine a subpart such as a page number, section name, etc. with the core URI in order to form a meaningful URL. An optional role can also be specified.
