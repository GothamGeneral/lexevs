package
nameToCode(
   String name,
   String codingScheme,
   CodingSchemeVersionOrTag versionOrTag):String		 Sequential
Tags: throws=LBException
Notes: Return the concept code that corresponds to the given concept name and
scheme.
<p>
If the terminology supports the "CONCEPT_NAME" property, this is
the property that will be looked up. If the CONCEPT_NAME property is
unique (as is the case with the NCI Thesaurus) then this method will
always map a particular name to a particular code.
<p>
If the code system does not have a "CONCEPT_NAME" property, then this
method does an exact match to a preferred presentation. If multiple
concept codes have the same presentation, an arbitrary code is returned.

@return The corresponding code; null if the name cannot be mapped.
